ispy-webgl v2.0.0 Release Notes

Highlights
----------
This is a major release that modernizes the entire build system and upgrades
core dependencies, bringing ispy-webgl into the modern JavaScript ecosystem.

Breaking Changes
----------------
- jQuery removed — the entire jQuery dependency has been removed and replaced
  with vanilla JS
- Bootstrap upgraded from 3 to 5 — UI components now use Bootstrap 5
- ES Modules & Vite — the project has been converted from vendored script tags
  to ES modules with a Vite-based build system; vendored libraries replaced
  with npm packages
- SVGRenderer removed — no longer supported as a rendering path

Major Upgrades
--------------
- Three.js upgraded from r125 to r182 — a significant jump spanning many
  upstream releases
- JSZip upgraded from 2.x to 3.8.0

New Features
------------
- GLTF/GLB import support — users can now import local GLTF and GLB files
- Transparency controls for GLTF files — opacity sliders now work with
  imported glTF geometries
- Config colors applied to glTF geometries
- AK4 and AK8 PF and PAT jet support (closes #188)
- CSC plus/minus geometries (closes #186)
- Packed candidate towers
- Display height slider — adjustable viewport height via slider
- Cursor feedback — cursor changes when highlighting rows in the event table

Bug Fixes
---------
- Fix muon-rphi-plus geometry (closes #182)
- Fix render order for imported geometries and physics objects
- Fix transparency behavior for muon chambers
- Remove missing sourcemap reference from bootstrap.min.css
- Remove jet radius scaling

Infrastructure
--------------
- Added production build with static asset copying
- Added installation and run instructions (INSTALL-AND-RUN.md)
- Updated license dates
